# Impact of Advanced Metrics on MLB All-Star Selections

## Overview
This project analyzes MLB All-Star hitter performance data to evaluate how the introduction of advanced baseball metrics influenced player evaluation and selection trends.

## Business Question
Did the increased use of advanced metrics change the type of hitters selected as MLB All-Stars?

## Data & Methodology
- Analyzed historical All-Star hitter performance data
- Compared traditional statistics (AVG, HR, RBI) with advanced metrics (OPS, OBP, SLG)
- Evaluated trends across multiple seasons to identify shifts in selection patterns
- Used Excel-based analysis and visualization to summarize findings

## Tools Used
- Microsoft Excel
- Data cleaning and validation
- Descriptive statistics
- Data visualization

## Key Takeaways
- Increased emphasis on efficiency-based metrics such as OPS and OBP
- Reduced reliance on raw counting statistics alone
- Evidence of a shift toward analytics-driven evaluation criteria

## Use Case
This analysis mirrors real-world decision-making scenarios where evolving performance metrics influence selection, evaluation, and strategy.
